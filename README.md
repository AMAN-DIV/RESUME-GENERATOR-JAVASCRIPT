**Online Resume Generator**

** Description**
This project is a user-friendly Online Resume Generator built with HTML, CSS, and JavaScript, utilizing the Bootstrap framework for enhanced responsiveness and styling. Empower users to effortlessly create professional resumes by filling in their details through a visually appealing and intuitive interface.

** Features**
- **User-Friendly Interface:** Intuitive design for easy navigation and data input.
- **Responsive Design:** Bootstrap framework ensures a seamless experience across various devices and screen sizes.
- **Dynamic Preview:** Real-time preview of the resume as users input their information.
- **Export Options:** Allow users to export their resumes in different formats, such as PDF or HTML.
- **Customization:** Incorporate various templates and styling options for personalized resumes.
- **Save and Edit:** Enable users to save their progress and return later for further editing.

 **Technologies Used**
- **HTML:** Structuring the content of the resume.
- **CSS:** Styling the layout and enhancing visual appeal.
- **JavaScript:** Adding dynamic behavior for an interactive experience.
- **Bootstrap:** Ensuring a responsive and modern design.

**How to Use**
1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser.
3. Fill in the required information in the provided fields.
4. Preview the resume in real-time.
5. Choose export options to save the resume in your preferred format.
